TOURNAMENT RULES

GENERAL RULES

Game Offline - Turnamen dilaksanakan secara offline di lokasi yang ditentukan panitia.

Peserta wajib membawa Device Mobile Phone (Android/iOS) sendiri.

Akun peserta harus sudah terdaftar di Panitia.

Waktu pertandingan ditentukan berdasarkan kesepakatan bersama tim dan panitia.

Kapten wajib Screenshot Room dan Hasil Pertandingan.

GAME RULES

Custom Draft Pick - Game mode tournament draft.

1x Game Pause diperbolehkan per match.

Ganti Akun/Nickname (OFF) - Role Roster (LOCK).

Chat All dan Radio All (OFF), Hero/Skin Trial, Emot (ON).

Game Remake Jika Situasi Mendesak (Sesuai Draft Pick Sebelumnya).
